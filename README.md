# K-Sparse Chaos Autoencoder

Разреженные автоэнкодеры теряют вместимость¹ на данных с высокой энтропией: регуляризация
по L1 приводит к полному коллапсу нейронов, TopK+ReLU — к 35% мёртвых нейронов. Данный
репозиторий содержит черновик статьи и все экспериментальные ноутбуки для метода,
устраняющего гибель нейронов через активации без фиксированных точек
f(x) = sin(αx) + β·tanh(γx) в сочетании с механизмом TopK (в данной работе α=8, β=0.5, γ=4).

## Основные результаты

- **0% мёртвых нейронов** на всех протестированных доменах (против 35–66% у стандартных активаций)
- **Улучшение дисперсии в 6.75×** над плотной ReLU базовой архитектурой (p<0.000001, N=10)
- **Монотонный контроль по K**: дисперсия гладко масштабируется с параметром разреженности (0 нарушений, K=4..112)
- Валидация на 4 доменах: логистическая карта, карта Хенона, Лоренц-96, MNIST

### Рисунки (`images/`)

Генерируются экспериментальными ноутбуками. Включают графики абляции активаций,
кривые K-разреженности, графики стабильности обучения, мультидоменную валидацию,
реконструкции MNIST и тепловые карты абляции коэффициентов.

### Сырые данные (`jsons/`)

JSON-файлы с полными результатами экспериментов — абляция активаций,
перебор коэффициентов, масштабирование данных, Лоренц-96, MNIST
и измерения дивергенции временного хаоса.

---

¹ **Вместимость (capacity)** — способность латентного пространства использовать все доступные измерения с ненулевой дисперсией. При потере вместимости часть нейронов «умирает» (перестаёт активироваться), а дисперсия оставшихся падает.
